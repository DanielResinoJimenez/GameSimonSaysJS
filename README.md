# 🎮 Simon Says Game | Web Interactive Edition
A modern web-based implementation of the classic memory game Simon Says. This project focuses on DOM manipulation, asynchronous logic for light sequences, and responsive design.

# 📝 Overview
This application challenges players to memorize and repeat increasingly complex sequences of colors and sounds. It serves as a practical demonstration of Vanilla JavaScript capabilities for game logic and state management without external frameworks.

# 🛠️ Tech Stack
Frontend: HTML5 & CSS3 (Flexbox/Grid for layout).

Logic: JavaScript ES6+ (Event Listeners, SetTimeout/Async logic for sequences).

Design: Custom UI with a focus on dark mode and accessibility.

# 🌟 Key Features
Dynamic Difficulty: Game speed and sequence complexity increase as you progress.

Persistent State: Track your current points and player session in real-time.

Interactive UI: High-contrast visual feedback for each button press.

Game Ecosystem: Includes a footer navigation to other mini-games (Hangman, Snake, Tetris).
